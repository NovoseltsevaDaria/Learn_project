# Hotel Management System (Отель)

**Лабораторная работа №3**  
Двухсервисное веб-приложение для управления номерами отеля.

**Технологии:** FastAPI + Pydantic + SQLite (backend) + Flask (frontend)

---

## Описание предметной области

Система позволяет управлять номерами отеля:
- Просмотр списка всех номеров
- Добавление нового номера
- Редактирование существующего номера
- Удаление номера

### Сущность **Room** (Номер)

| Поле | Тип | Описание |
|------|-----|----------|
| `id` | INTEGER | Уникальный идентификатор (автоинкремент) |
| `room_number` | INTEGER | Номер комнаты (101, 102, ...) |
| `type` | TEXT | Тип: `single`, `double`, `suite` |
| `price_per_night` | REAL | Цена за ночь в рублях |
| `is_available` | BOOLEAN | Доступен ли номер (True/False) |
| `capacity` | INTEGER | Вместимость (1–6 человек) |
| `description` | TEXT | Дополнительное описание (опционально) |

## Архитектура проекта
    Браузер (пользователь)
                │
                ↓ (HTTP)
    Flask (фронтенд, порт 5000)
                │
                ↓ (HTTP requests)
    FastAPI (бэкенд, порт 8000)
                │ (SQL)
    SQLite (файл hotel.db)
